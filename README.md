# snippets-server
Node js server. Mongo DB,Express, Rest API For app.

Database Snippets:

Collections:

    1.categories:
        - name.
    2.subcategories:
        - category;
        - name.
    3.Products:
        - subcategory;
        - name.
    4.tags:
        - productId;
        - html;
        - css;
        - js;
    5. comments:
        - productId;
        - name;
        - text.

Requests:

    1. Categories:
        - get;
        - post;
        - put(Id);
        - delete(Id.
    2. Products:
        - get;
        - post;
        - put(Id);
        - delete(Id).
    3. Subcategory:
        - get;
        - post;
        - put(Id);
        - delete(Id).
